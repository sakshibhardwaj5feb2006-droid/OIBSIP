# Personal Portfolio Website

## 📌 Description

This is a modern and responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript. The portfolio showcases my profile, education, technical skills, projects, and contact information in a professional and visually appealing way. It also includes a resume download feature and interactive animations inspired by modern portfolio designs.

## ✨ Features

* Responsive Design
* Modern Dark-Themed User Interface
* Hero Section with Personal Introduction
* Animated Profile Section
* About Me Section
* Education Section
* Technical Skills Section
* Projects Showcase
* Contact Information with Clickable Links
* Resume Download Button
* Smooth Scrolling and Interactive Elements

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Add your images (`babe1.jpeg` and `babe2.jpeg`) in the project folder.
4. Add your `resume.pdf` file in the project folder.
5. Open `index.html` in your browser or run it using Live Server.
