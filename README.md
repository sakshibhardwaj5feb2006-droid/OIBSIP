# Personal Portfolio Website

## 📌 Description

This is a responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript. It showcases my skills, projects, education, and contact information.

## ✨ Features

* Responsive Design
* Modern User Interface
* About Me Section
* Skills Section
* Projects Section
* Contact Information
* Smooth Scrolling using JavaScript

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
