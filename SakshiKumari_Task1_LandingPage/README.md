# 🏋️ FitZone Gym - Landing Page

## 📖 Description

FitZone Gym is a modern and responsive landing page designed for a fitness center. It showcases gym services, professional trainers, workout programs, membership benefits, and fitness statistics in an attractive user interface.

## ✨ Features

- Responsive Design
- Modern Navigation Bar
- Hero Section with Call-to-Action Button
- Professional Trainers Section
- Modern Equipment Section
- Benefits Cards
- Statistics Section
- Footer with Contact Information
- Mobile Friendly Layout

## 🛠️ Technologies Used

- HTML5
- CSS3
