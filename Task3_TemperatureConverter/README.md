# Temperature Converter

## Description
A Temperature Converter Web Application developed using HTML, CSS and JavaScript.

## Features
- Celsius to Fahrenheit and Kelvin
- Fahrenheit to Celsius and Kelvin
- Kelvin to Celsius and Fahrenheit
- Responsive UI

## Technologies Used
- HTML
- CSS
- JavaScript
